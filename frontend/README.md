# Project Title Database webpage

I was making a webpage that goes along with a database I made this semester.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Use vscode built in features to set up node.js and react pages
2. Modify react pages from template to suit my needs with html, js, and css.
3. Set up backend to import from database and get rid of the failsafe code.

Instructions for using the software:

1. Open up the read me file
2. run the directed commands, first the dependencies and then the back end and front end in separate teminals
⚠️Has to have their own database connected ⚠️
